# Right Size with Sena

Professional website for a Senior Move & Right-Sizing Specialist business. Helps older adults navigate downsizing and moving with compassion and support.

## Site Structure

A multi-page website showcasing services for senior relocation and right-sizing:

### Pages

1. **Home** (`index.html`) - Hero section with business introduction and value propositions
2. **About Sena** (`about.html`) - Personal background, approach, and core values
3. **Services** (`services.html`) - Detailed service offerings and flexible packages
4. **How It Works** (`how-it-works.html`) - Step-by-step process from consultation to settling in
5. **Contact** (`contact.html`) - Contact form, information, and FAQ

## Features

- Clean, professional design with warm color palette
- Fully responsive mobile-friendly layout
- Emoji-based visual elements for warmth and accessibility
- Smooth navigation between pages
- Contact form and information gathering
- FAQ section for common questions
- Service packages and pricing information

## Design

- **Color Scheme**: Professional blues (#2c3e50) with warm accent (#e74c3c)
- **Typography**: Montserrat font family for modern, clean look
- **Layout**: Grid-based responsive design that adapts to all screen sizes

## Running the Site

```bash
cd ~/sena-portfolio
python3 -m http.server 8000
# Visit http://localhost:8000
```

## Files

- `index.html` - Home page
- `about.html` - About Sena page
- `services.html` - Services overview
- `how-it-works.html` - Process explanation
- `contact.html` - Contact and FAQ
- `styles.css` - Responsive styling for all pages
- `scripts.js` - Smooth scroll navigation
- `assets/` - Image folder for photos

## Customization

- Update contact email in `contact.html`
- Replace emoji placeholders with actual images in `/assets/`
- Modify color variables in CSS `:root` section
- Add phone number and other contact details as needed
